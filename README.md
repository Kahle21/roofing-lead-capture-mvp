# Roofing Lead Capture MVP

FastAPI + Twilio + Supabase MVP for roofing companies.

## Goal
Capture missed calls, auto-text the caller, collect lead details by SMS, and notify the owner.

## Stack
- FastAPI
- Twilio
- Supabase
- Jinja2
- Render
- ngrok for local webhook testing

## Local setup
1. Create venv
2. Install requirements
3. Fill .env
4. Run uvicorn
5. Expose with ngrok
